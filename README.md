# Date Fruit Multiclass Classification Using Artificial Neural Networks

## Objective

The objective of this project is to classify seven different date fruit varieties using an Artificial Neural Network (ANN) model based on morphological and color-based features.

## Dataset

Date Fruit Dataset

- 898 samples
- 34 input features
- 7 classes:
  - Barhee
  - Deglet Nour
  - Sukkary
  - Rotab Mozafati
  - Ruthana
  - Safawi
  - Sagai

## Project Workflow

1. Data Exploration
2. Data Preprocessing
3. Feature Scaling
4. ANN Model Development
5. Model Training
6. Performance Evaluation
7. Results Analysis
